# Circle Suvivor

Page: https://epharia.github.io/mobile/

***

## About

A simple ***Bullet Hell*** suvivor game.

## Controls

| **Action** | **Desktop** | **Mobile** |
| --- | --- | --- |
| Move | *'Arrow-Keys' or WASD* | *Virtual Joystick* (Left) |
| Aim | *Mouse Position* | *Virtual Joystick* (Right) |
| Pause | *'Esc-Key'* | *Tap Screen* |

*the controls are likely to be changed*
